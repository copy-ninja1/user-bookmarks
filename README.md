## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Install NPM packages

```bash
npm install
```

## Install Docker

[documentation](https://docs.docker.com/engine/install/)

## Start Up Container

```bash
docker compose up dev-db
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
